# A simple Tictactoe Game built with Expo React-native

### Google Play Store Url
* [Install Tictactoe](https://play.google.com/store/apps/details?id=com.mecitdeniz.tictactoe)

## Features
* Multiplayer
* Singleplayer
* Online (soon)
* Emojis
### Support multiple languages
* German
* Russian
* English
* Turkish
* Polish
* French
* Spanish
